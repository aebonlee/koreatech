# 프로젝트 평가 기준

## 기능 완성도

### 필수 기능
- [x] 홈 페이지 (Hero + 과목 소개 + 커리큘럼 카드)
- [x] 파이썬 기초 강의 페이지 (변수, 자료형, 입출력)
- [x] 제어문 강의 페이지 (if/for/while)
- [x] 함수 강의 페이지 (def, lambda, scope)
- [x] 자료구조 강의 페이지 (list, tuple, dict, set)
- [x] 게시판 목록 (카테고리 필터, 페이지네이션)
- [x] 게시글 상세 (내용 + 댓글)
- [x] 게시글 작성 (로그인 필요)
- [x] 로그인/회원가입 (Google, Kakao, Email)
- [x] 마이페이지 (프로필 수정)
- [x] 다크모드/컬러테마 전환
- [x] 한/영 언어 전환
- [x] 모바일 반응형

### 선택 기능
- [x] 검색 모달
- [x] AOS 애니메이션
- [x] SEO 메타 태그
- [x] 404 페이지
- [x] 비밀번호 찾기

## 코드 품질
- React 19 최신 기능 활용
- 컴포넌트 Lazy Loading (코드 스플리팅)
- Context API를 활용한 상태 관리
- Supabase 미설정 시 안전한 폴백 처리
- CSS 변수 기반 테마 시스템

## 접근성
- ARIA 레이블 적용
- 키보드 네비게이션 지원 (ESC로 모달 닫기)
- 시맨틱 HTML 구조

## 성능
- Vite 7 빌드 최적화
- 이미지 lazy loading
- CSS gzip 14KB, JS gzip 84KB
