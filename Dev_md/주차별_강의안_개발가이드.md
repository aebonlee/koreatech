# 주차별 강의안 기능 개발 가이드

## 개요
KoreaTech 컴퓨팅 사고 교육 사이트에 **주차별 강의안** 기능을 구현합니다.
- 외부 슬라이드 iframe 임베드 + 텍스트 설명 + 첨부파일 링크 지원
- 관리자만 등록/수정/삭제 가능 (학생은 열람만)
- 주차 수 제한 없이 자유 등록

## 파일 구조

### 신규 파일
| 파일 | 설명 |
|------|------|
| `src/pages/Lectures.jsx` | 카드 그리드 목록 페이지 (`/lectures`) |
| `src/pages/LectureDetail.jsx` | 상세 페이지 - iframe + 본문 + 첨부 (`/lectures/:id`) |
| `src/pages/LectureWrite.jsx` | 등록/수정 폼 - 관리자 전용 (`/lectures/write`, `/lectures/edit/:id`) |

### 수정 파일
| 파일 | 변경 내용 |
|------|-----------|
| `src/utils/supabase.js` | 강의안 CRUD 함수 6개 추가 |
| `src/config/site.js` | menuItems, footerLinks에 강의안 메뉴 추가 |
| `src/layouts/PublicLayout.jsx` | lazy import 3개 + Route 4개 추가 |
| `src/utils/translations.js` | `site.lectures.*` 번역 키 추가 (ko/en) |
| `src/styles/site.css` | 강의안 카드/상세/폼 스타일 + 다크모드 + 반응형 추가 |

## Supabase 함수 (supabase.js)

| 함수 | 설명 |
|------|------|
| `getLectures()` | 공개 강의안 목록 (week_number ASC) |
| `getLectureById(id)` | 강의안 상세 |
| `createLecture(data)` | 강의안 등록 |
| `updateLecture(id, data)` | 강의안 수정 |
| `deleteLecture(id)` | 강의안 삭제 |
| `incrementLectureViews(id)` | 조회수 증가 (RPC) |

## 라우팅

| 경로 | 컴포넌트 | 권한 |
|------|----------|------|
| `/lectures` | `Lectures` | 공개 |
| `/lectures/write` | `LectureWrite` | AuthGuard (관리자) |
| `/lectures/edit/:id` | `LectureWrite` | AuthGuard (관리자) |
| `/lectures/:id` | `LectureDetail` | 공개 |

## lectures 테이블 스키마

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `id` | BIGSERIAL PK | 자동 증가 ID |
| `week_number` | INTEGER NOT NULL | 주차 번호 |
| `title` | TEXT NOT NULL | 강의 제목 |
| `description` | TEXT | 간단 설명 (카드에 표시) |
| `content` | TEXT | 상세 내용 |
| `slide_url` | TEXT | 슬라이드 iframe URL |
| `file_url` | TEXT | 첨부파일 다운로드 URL |
| `cover_image` | TEXT | 카드 커버 이미지 URL |
| `tags` | TEXT[] | 태그 배열 |
| `is_published` | BOOLEAN | 공개 여부 (기본 true) |
| `views` | INTEGER | 조회수 (기본 0) |
| `user_id` | UUID FK | 작성자 (auth.users) |
| `created_at` | TIMESTAMPTZ | 생성일 |
| `updated_at` | TIMESTAMPTZ | 수정일 |

## CSS 클래스

### 카드 그리드
- `.lecture-grid` — 3열 → 2열(1024px) → 1열(768px) 반응형
- `.lecture-card` — hover translateY + shadow
- `.lecture-card-image` — 16:9 비율 커버
- `.lecture-week-badge` — 주차 배지 (curriculum-step 패턴)
- `.lecture-tag` — 태그 칩

### 상세 페이지
- `.lecture-detail` — 카드 레이아웃
- `.lecture-slide-container` — 16:9 iframe (padding-bottom: 56.25%)
- `.lecture-detail-content` — 텍스트 본문
- `.lecture-file-section` — 첨부파일 다운로드

### 폼
- `.lecture-write-form` — 등록/수정 폼
- `.form-row` — 주차+제목 가로 배치
- `.form-checkbox-label` — 공개 여부 체크박스

## 검증 방법
1. `npm run build` 빌드 성공
2. 네비게이션에 "주차별 강의안" 메뉴 표시
3. `/lectures` 카드 그리드 렌더링 (빈 상태 메시지)
4. `/lectures/write` 비관리자 접근 거부, 관리자 폼 표시
5. 다크모드/모바일 반응형 확인
6. Supabase 테이블 생성 후 CRUD 동작 확인
