# KoreaTech 컴퓨팅 사고 교육 사이트 - 개발일지 v8

## 2026-03-03 - 강의안 목록형 전환 + 강의자료 버튼 정리

### 작업 내용

#### 1. 주차별 강의안 페이지 — 카드 그리드 → 목록형 테이블 전환

`/lectures` 페이지를 카드 그리드 레이아웃에서 Board.jsx와 동일한 **목록형 테이블** 레이아웃으로 전환.

**Before:** `.lecture-grid` > `.lecture-card` (카드 이미지+본문)
**After:** `.board-table-wrapper` > `table.board-table` (목록 테이블)

**테이블 컬럼:**
| 컬럼 | 내용 |
|------|------|
| 주차 | `lecture-week-badge` (Week N) |
| 제목 | 클릭 시 → `/lectures/:id` 상세 이동 |
| 설명 | description (모바일 숨김) |
| 등록일 | created_at |
| 조회수 | views |

#### 2. 강의자료 다운로드 페이지 — 버튼 정리

`/lectures/materials` 페이지에서 기존 3개 버튼(보기/새탭/다운로드)을 2개로 정리:

| Before | After |
|--------|-------|
| 보기 (iframe 뷰어) | 보기 (새 탭에서 PDF 열기) |
| 새탭 | 제거 (보기에 통합) |
| 다운로드 | 다운로드 (유지) |

- iframe PDF 뷰어 제거 (불필요)
- `useState` import 제거
