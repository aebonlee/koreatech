# KoreaTech 컴퓨팅 사고 교육 사이트 - 개발일지 v11

## 2026-03-03 - 커뮤니티 메뉴 확장 + 갤러리 웹링크 + 수정 기능

### 작업 배경

- 상위 메뉴 "게시판"을 "커뮤니티" 드롭다운으로 확장 (게시판 + 갤러리 + 포트폴리오)
- URL 체계를 `/community/*`로 통일하고 기존 `/board` 리다이렉트 처리
- 갤러리에 웹 링크(link_url) 필드 추가 (이미지 URL과 별도)
- 갤러리/포트폴리오 수정(edit) 기능 추가

### 변경 사항

#### 1. 네비게이션 변경 (`site.js`)

| Before | After |
|--------|-------|
| 게시판 (단일 메뉴) | 커뮤니티 (드롭다운: 게시판 / 갤러리 / 포트폴리오) |

- footerLinks에도 갤러리, 포트폴리오 추가

#### 2. 새 페이지 6개 생성

| 파일 | 설명 |
|------|------|
| `Gallery.jsx` | 갤러리 목록 — 카테고리 필터(전체/작품/프로젝트/스크린샷), 이미지 카드 그리드 |
| `GalleryDetail.jsx` | 갤러리 상세 — 이미지, 웹 링크 버튼, 댓글, 수정/삭제 버튼 |
| `GalleryWrite.jsx` | 갤러리 등록/수정 — 작품명, 카테고리, 이미지 URL, 웹 링크, 설명 |
| `Portfolio.jsx` | 포트폴리오 목록 — 블로그형 카드(커버이미지+제목+요약+태그) |
| `PortfolioDetail.jsx` | 포트폴리오 상세 — 커버 이미지, 태그, 본문, 댓글, 수정/삭제 버튼 |
| `PortfolioWrite.jsx` | 포트폴리오 등록/수정 — 제목, 요약, 커버 이미지, 태그, 내용 |

#### 3. 라우트 변경 (`PublicLayout.jsx`)

| 경로 | 컴포넌트 | 비고 |
|------|----------|------|

| `/community/gallery/:id` | `GalleryDetail` | 갤러리 상세 |
| `/community/portfolio` | `Portfolio` | 포트폴리오 목록 |
| `/community/portfolio/write` | `PortfolioWrite` | 포트폴리오 등록 |
| `/community/portfolio/edit/:id` | `PortfolioWrite` | 포트폴리오 수정 |
| `/community/portfolio/:id` | `PortfolioDetail` | 포트폴리오 상세 |
| `/board` → `/community/board` | Navigate | 리다이렉트 |

- **Portfolio**: getPortfolios, getPortfolioById, createPortfolio, **updatePortfolio**, deletePortfolio, incrementPortfolioViews, getPortfolioComments, createPortfolioComment, deletePortfolioComment
