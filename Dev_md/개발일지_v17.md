# KoreaTech 컴퓨팅 사고 교육 사이트 - 개발일지 v17

## 2026-03-03 - 모바일 테이블 스크롤 최적화 및 가독성 개선

### 변경 사항

#### 1. 테이블 좌우 스크롤 + < > 화살표 표시

모바일에서 테이블이 화면을 넘칠 때 좌우 스크롤이 가능하도록 개선하고,
파란색 < > 원형 버튼으로 스크롤 방향을 안내.

| 대상 | 처리 방식 |
|------|-----------|
| `.lesson-table` (교육 콘텐츠 전체) | 자동으로 스크롤 컨테이너(`table-scroll-container`)로 래핑 |
| `.board-table-wrapper` (게시판, 주차별 계획 등) | 기존 래퍼에 화살표 추가 |
| `.materials-table-wrapper` (강의자료) | 기존 래퍼에 화살표 추가 |

- `useCodeCopy` 패턴과 동일한 MutationObserver 방식으로 전역 자동 적용
- 스크롤 위치에 따라 좌/우 화살표가 동적으로 표시/숨김
- ResizeObserver로 화면 크기 변경 시에도 정상 동작

#### 2. 모바일 가독성 개선

| 항목 | 768px 이하 | 480px 이하 |
|------|------------|------------|
| `.lesson-table` 셀 | padding 8px 12px, font 0.85rem | padding 6px 10px, font 0.8rem |
| `.lesson-body p` | font 0.95rem, line-height 1.9 | font 0.9rem, line-height 1.95 |
| `.lesson-body li` | font 0.95rem, margin 6px | font 0.9rem |
| `.lesson-body h3` | font 1.15rem | font 1.1rem |
| `.callout-box` | padding 16px 18px, font 0.9rem | padding 14px 16px, font 0.85rem |
| `.code-block` | — | margin 확장, 모서리 축소 |
