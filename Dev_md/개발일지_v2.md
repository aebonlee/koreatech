### 4. 파이썬 강의 콘텐츠 대폭 확장 (아카데믹 수준)

각 주차별 페이지를 대학교 2시간 수업에 맞는 학습자료 수준으로 재작성:

**콘텐츠 구조 (매 주차 공통):**
- 학습 목표 (`callout-box`) — 해당 주차 핵심 학습목표 4-6개
- 상세 이론 — h2/h3 구분, 역사적 배경 포함
- 코드 예제 — `code-block` + `code-header`, 실행 결과 포함
- 비교 표 — `lesson-table`로 구조화
- 핵심 정리 (`callout-box`) — 해당 주차 요약
- 자주 하는 실수 (`callout-box`) — 초보자 함정
- 실습 과제 (`exercise-box`) — 기본 5-6문제
- 심화 문제 (`exercise-box`) — 도전 2-3문제
- 주차별 네비게이션 (`lesson-nav`) — 이전/다음 주차 이동
