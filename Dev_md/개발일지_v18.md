## 2026-03-04 ~ 2026-03-05 - OG 메타 태그 보완 및 localStorage 제거

### 변경 사항

#### 1. Open Graph 메타 태그 보완 (카카오톡 공유 최적화)

웹 URL 공유 시 미리보기 정보가 정확하게 표시되도록 OG 메타 태그 보완.

| OG 태그 | 값 |
|---------|-----|
| `og:url` | `https://koreatech.dreamitbiz.com` |
| `og:title` | `KoreaTech 컴퓨팅 사고 \| 한국기술교육대학교` |
| `og:description` | `파이썬으로 시작하는 컴퓨팅 사고 교육 - 파이썬 기초, AI 활용 가이드, 실습 자료까지 한곳에서` |
| `og:type` | `website` |
| `og:image` | `https://koreatech.dreamitbiz.com/og-image.png` |
| `og:image:width` | `400` (신규) |
| `og:image:height` | `302` (신규) |
| `og:site_name` | `KoreaTech 컴퓨팅 사고` |
| `og:locale` | `ko_KR` (신규) |

- `index.html`: description 개선, og:locale·og:image:width/height 추가, twitter:title/description 추가
- `SEOHead.jsx`: 동일하게 og:locale·og:image:width/height 추가, 기본 description 개선

#### 2. localStorage 사용 전면 제거

모든 데이터의 로컬스토리지 사용을 금지하는 정책에 따라, `ThemeContext.jsx`에서 localStorage 6건 모두 제거.

| 항목 | 이전 (localStorage) | 변경 후 |
|------|---------------------|---------|
| `themeMode` | `getItem/setItem` | 기본값 `'auto'` (시간 기반 자동) |
| `colorTheme` | `getItem/setItem` | 기본값 `'blue'` |
| `theme` (레거시) | `removeItem` | 삭제 |

- 전체 프로젝트 localStorage/sessionStorage 사용 **0건** 확인
