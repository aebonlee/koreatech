### 1. 코드 블록 복사(Copy) 버튼 추가

**커밋:** `9bcb825`

모든 `.code-block`에 자동으로 Copy 버튼을 삽입하는 전역 훅 구현.

**신규 파일:**
- `src/hooks/useCodeCopy.js` — MutationObserver 기반 전역 훅

**동작 방식:**
- `PublicLayout.jsx`에서 `useCodeCopy()` 호출
- DOM에 `.code-block`이 렌더링되면 자동으로 Copy 버튼 삽입
- `code-header`가 있는 경우 → 헤더 우측에 배치
- `code-header`가 없는 경우 → 블록 우측 상단에 absolute 배치
- Clipboard API로 `<pre>` 텍스트 복사
- 클릭 시 "Copied!" 피드백 (2초간 초록색 표시)
- MutationObserver로 lazy 로드 페이지에도 자동 대응

**CSS 추가 (`site.css`):**
- `.code-copy-btn` — 투명 배경, 흰색 테두리, hover 시 밝아짐
- `.code-copy-btn.copied` — 초록색(#68d391) 테두리+텍스트
- `.code-header` — `display: flex; justify-content: space-between` 으로 변경

**수정 파일:**
| 파일 | 변경 내용 |
|------|----------|
| `src/hooks/useCodeCopy.js` | 신규 — 전역 Copy 버튼 훅 |
| `src/layouts/PublicLayout.jsx` | `useCodeCopy` import 및 호출 추가 |
| `src/styles/site.css` | `.code-copy-btn` 스타일, `.code-header` flex 레이아웃 |

### 2. 서브페이지 타이틀 헤더 스타일 추가 (page-header)

**커밋:** `b36e354`

dreamitbiz.com 본사이트 스타일을 참고하여 모든 서브페이지에 그라데이션 배경 타이틀 헤더 적용.

**기존 문제:**
- `.page-header` 클래스가 이미 모든 서브페이지에 사용되고 있었으나 기본 CSS 정의가 없었음
- 반응형(responsive.css)과 다크모드(dark-mode.css) 규칙만 존재

**추가된 기본 스타일 (`site.css`):**
```css
.page-header {
  padding: calc(var(--nav-height) + 48px) 0 48px;
  background: var(--hero-bg);    /* 135도 3단 그라데이션 */
  color: #fff;
  text-align: center;
}
.page-header h1 { font-size: 2.2rem; font-weight: 800; }
.page-header p  { font-size: 1.05rem; opacity: 0.85; }
```

**자동 대응:**
- 5색 컬러테마: `--hero-bg` 변수가 blue/red/green/purple/orange별로 정의됨
- 다크모드: `[data-theme="dark"] .page-header { background: var(--hero-bg-dark); }` (기존)
- 반응형: 태블릿(36px)/모바일(24px) 폰트 크기 조절 (기존 responsive.css)

**적용 범위 (기존 `.page-header` 사용 페이지 전체):**
- `/python/*` (PythonLesson, PythonWeek1~11)
- `/ct/*` (WeeklyPlan, Syllabus, PracticeGuide)
- `/board`, `/board/:id`, `/board/write`
- `/lectures`, `/lectures/:id`, `/lectures/write`
- `/mypage`

**수정 파일:**
| 파일 | 변경 내용 |
|------|----------|
| `src/styles/site.css` | `.page-header`, `.page-header h1`, `.page-header p` 기본 스타일 추가 |
