# KoreaTech 컴퓨팅 사고 교육 사이트 - 개발일지 v16

## 2026-03-03 - 모바일 히어로 타이틀 및 로고 2줄 표시

### 변경 사항

#### 1. 모바일 히어로 타이틀 줄바꿈

히어로 영역 타이틀을 모바일(≤480px)에서 2줄로 표시되도록 변경.

| 화면 | 표시 |
|------|------|
| 데스크톱/태블릿 | `파이썬으로 시작하는 컴퓨팅 사고` (한 줄) |
| 모바일 (≤480px) | `파이썬으로 시작하는` / 줄바꿈 / `컴퓨팅 사고` |

- `Home.jsx`에서 타이틀을 `<span>` 2개로 분리
- `site.css` 480px 미디어쿼리에 `.hero-title-line2 { display: block }` 추가

#### 2. 모바일 로고 2줄 표시

네비게이션 로고를 모바일에서 2줄로 표시되도록 변경.

| 화면 | 표시 |
|------|------|
| 데스크톱/태블릿 | `KoreaTech CT` (한 줄) |
| 모바일 | `KoreaTech` / 줄바꿈 / `CT` |

- `responsive.css`에 `.logo .brand-biz { display: block }` 추가
- `.logo h1`에 `line-height: 1.2` 추가
