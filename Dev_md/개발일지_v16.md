# KoreaTech 컴퓨팅 사고 교육 사이트 - 개발일지 v16

## 2026-03-03 - 모바일 히어로 타이틀 줄바꿈 및 로고 정리

### 변경 사항

#### 1. 모바일 히어로 타이틀 줄바꿈

히어로 영역 타이틀을 모바일(≤480px)에서 2줄로 표시되도록 변경.

| 화면 | 표시 |
|------|------|
| 데스크톱/태블릿 | `파이썬으로 시작하는 컴퓨팅 사고` (한 줄) |
| 모바일 (≤480px) | `파이썬으로 시작하는` / 줄바꿈 / `컴퓨팅 사고` |

- `Home.jsx`에서 타이틀을 `<span>` 2개로 분리
- `site.css` 480px 미디어쿼리에 `.hero-title-line2 { display: block }` 추가

#### 2. 모바일 로고 — CT를 로고에 포함 (한 줄 유지)

모바일에서도 로고를 `KoreaTech CT` 한 줄로 유지.

| 화면 | 표시 |
|------|------|
| 모든 화면 | `KoreaTech CT` (한 줄) |

### 수정 파일

| 파일 | 변경 내용 |
|------|-----------|
| `src/pages/Home.jsx` | 히어로 타이틀 `<span>` 2개로 분리 |
| `src/styles/site.css` | 480px 이하 `.hero-title-line2` block 처리 |
| `src/styles/responsive.css` | 모바일 로고 한 줄 유지 |
