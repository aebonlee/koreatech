# KoreaTech 컴퓨팅 사고 교육 사이트 - 개발일지 v19

## 2026-03-12 - 참고자료 페이지 UI 개선 (드롭다운 + 보기/다운로드 버튼)

### 변경 사항

#### 1. 참고자료(/references) 페이지 UI 전면 개편

기존 게시판형 목록에서 **강의자료(/lectures) 스타일**의 테이블 UI로 변경.
행 클릭 시 드롭다운으로 상세 내용을 펼쳐 볼 수 있고, 첨부파일에 대한 보기/다운로드 버튼 제공.

| 항목 | 이전 | 변경 후 |
|------|------|---------|
| 테이블 스타일 | `board-table` (기본 게시판) | `materials-table` (강의자료 스타일) |
| 제목 클릭 | 상세 페이지로 이동 | 드롭다운으로 내용 펼침 |
| 첨부파일 | 상세 페이지에서만 다운로드 | 목록에서 바로 보기/다운로드 |
| 날짜/조회수 컬럼 | 별도 컬럼 표시 | 제거 (간결한 UI) |
| 펼침 아이콘 | 없음 | ▶ 화살표 (펼치면 90도 회전) |

#### 2. 주요 기능

- **드롭다운 아코디언**: 행 클릭 시 내용(content)을 드롭다운으로 표시
- **보기 버튼**: 첨부파일을 새 탭에서 열기 (로그인 필요)
- **다운로드 버튼**: 첨부파일 직접 다운로드 (로그인 필요)
- **관리자 수정 버튼**: 관리자에게만 노란색 수정 버튼 표시
- **파일 없음 표시**: 첨부파일이 없는 항목은 "파일 없음" 텍스트
- **조회수 자동 증가**: 드롭다운 펼칠 때 조회수 증가 처리

#### 3. 수정 파일

| 파일 | 변경 내용 |
|------|-----------|
| `src/pages/Lectures.jsx` | 전면 개편 - materials-table 스타일, 드롭다운 아코디언, 보기/다운로드 버튼 |
| `src/utils/translations.js` | `references.noFile`, `references.noContent` 번역키 추가 (한/영) |
| `src/styles/site.css` | 참고자료 드롭다운 CSS 추가 (ref-* 클래스, 반응형, 다크모드) |

#### 4. CSS 신규 클래스

| 클래스 | 용도 |
|--------|------|
| `.ref-col-actions` | 보기/다운로드 액션 컬럼 (260px) |
| `.ref-title-row` | 제목 + 펼침 아이콘 flex 행 |
| `.ref-expand-icon` | ▶ 아이콘 (펼치면 90도 회전 애니메이션) |
| `.ref-dropdown-row` | 드롭다운 내용 행 |
| `.ref-dropdown-content` | 드롭다운 내용 영역 (배경색 구분) |
| `.ref-content-text` | 본문 텍스트 스타일 |
| `.ref-edit-btn` | 관리자 수정 버튼 (노란색) |
| `.ref-no-file` | 파일 없음 텍스트 |

#### 5. 반응형 & 다크모드 지원

- **768px 이하**: 액션 컬럼 폭 자동 조정
- **다크모드**: 드롭다운 배경, 수정 버튼 등 모두 다크 테마 적용
