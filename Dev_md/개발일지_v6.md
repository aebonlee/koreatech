# KoreaTech 컴퓨팅 사고 교육 사이트 - 개발일지 v6

## 2026-03-03 - 강의자료 PDF 다운로드/뷰어 페이지 구현

### 작업 배경

기존에 `pdf/` 폴더에 있던 강의 PDF 파일(week01~12.pdf)이 Vite의 정적 파일 서빙 경로인 `public/` 밖에 위치하여 배포 시 404 발생. 강의자료를 목록으로 확인하고, 바로 보기/다운로드할 수 있는 전용 페이지가 필요했음.

### 작업 내용

#### 1. PDF 파일 이동

- `pdf/*.pdf` 12개 파일 → `public/pdf/`로 이동
- Vite 빌드 시 `public/` 하위 파일이 `dist/`로 자동 복사되므로 배포 시 포함
- 기존 `pdf/` 폴더 및 `public/pdf/.gitkeep` 제거

#### 2. 신규 페이지: 강의자료 다운로드 (`/lectures/materials`)

| 기능 | 설명 |
|------|------|
| PDF 목록 테이블 | 주차, 제목, 파일명, 액션 버튼 |
| 보기 버튼 | 페이지 하단 iframe으로 PDF 임베드 뷰어 토글 |
| 새 탭 버튼 | 브라우저 내장 PDF 뷰어로 열기 |
| 다운로드 버튼 | `<a download>` 속성으로 직접 다운로드 |
| 다국어 지원 | 한국어/영어 제목 및 UI 텍스트 전환 |

#### 3. 메뉴 구조 변경

기존 단일 링크 "주차별 강의안"을 드롭다운으로 변경:
- **강의자료 다운로드** (`/lectures/materials`) — 신규
- **강의안 게시판** (`/lectures`) — 기존

#### 4. 스타일 추가

- `.materials-table` — PDF 목록 테이블
- `.materials-btn` — 보기/새탭/다운로드 버튼 (각각 색상 구분)
- `.pdf-viewer` — iframe PDF 뷰어 컨테이너
- 다크모드 대응 (모든 컴포넌트)
- 반응형 대응 (768px, 480px 브레이크포인트)
