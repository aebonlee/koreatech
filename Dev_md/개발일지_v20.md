#### 2. GitHub Pages SPA 라우팅 404 오류 수정

`/references` 등 직접 URL 접근 시 GitHub Pages 404 오류 발생 문제 해결.

| 항목 | 내용 |
|------|------|
| 원인 | GitHub Pages는 SPA 클라이언트 라우팅을 지원하지 않아 실제 파일이 없는 경로에서 404 반환 |
| 해결 | `vite.config.js`에 빌드 후 `index.html` → `404.html` 자동 복사 플러그인 추가 |
| 원리 | GitHub Pages는 404 발생 시 `404.html`을 보여주고, 이 파일이 SPA 앱 자체이므로 React Router가 정상 라우팅 처리 |
