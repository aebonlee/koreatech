# KoreaTech 컴퓨팅 사고 교육 사이트 - 개발일지 v7

## 2026-03-03 - 강의자료 주차별 제목 수정

### 작업 배경

v6에서 구현한 강의자료 다운로드 페이지(`/lectures/materials`)의 1~12주차 제목을 실제 강의 커리큘럼에 맞게 수정.

### 변경된 주차별 제목

| 주차 | 이전 제목 | 변경 후 제목 |
|------|----------|-------------|
| 1 | 컴퓨팅 사고 개요 | 입력과 출력 (변수, print, input) |
| 2 | 입출력과 변수 | 데이터 연산 (계산기 만들기) |
| 3 | 자료형과 연산 | 데이터 표현 및 조작 (리스트) |
| 4 | IPO와 PDC | 문제 정의와 핵심아이디어 |
| 5 | 터틀 그래픽 | 컴퓨터로 이해하기 (터틀 그래픽) |
| 6 | 순서도 | 기능 단위 분리 (함수) |
| 7 | IF 조건문 | 알고리즘 표현 (순서도) |
| 8 | 반복문 | 조건문 실습 (if-else) |
| 9 | 함수 | 반복문 실습 (while, for) |
| 10 | 예외처리 | 알고리즘 심화 (정렬 등) |
| 11 | 리스트 | 에러와 디버깅 |
| 12 | 종합 정리 | 프로젝트 (포트폴리오 만들기) |
